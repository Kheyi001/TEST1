# Project Blueprint

## Overview

This project is a React application that allows users to browse and search for cars. It includes a login page and a car listing page.

## Implemented Features

### Login Page

*   A clean and modern login form.
*   Email and password input fields.
*   Password visibility toggle.
*   Consent checkbox.
*   "Connexion" and "Inscription" buttons.
*   "Mot de passe oubli√©" link.

### Car Listing Page

*   Displays a list of cars from a data object.
*   Shows the total number of search results.
*   Includes a sidebar for filtering results.
*   Each car is displayed in a card with its image, name, price, and other details.
*   Built with `react-bootstrap`.
